# To Do List

January 25, 2016

##### By Molly Curtin &amp; Sean Peerenboom

### Description

A web application that takes in tasks from a user and creates a to-do list and then adds them to the completed list as they are crossed off.

##### Technologies Used

* HTML
* CSS
* JavaScript
* jQuery
* Mocha
* Chai
* BDD
* Git
